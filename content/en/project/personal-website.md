This website is my personal academic portfolio, built as part of my studies.



\## Technologies Used

\- \*\*GitHub Pages\*\* - Free hosting

\- \*\*Markdown\*\* - Content creation

\- \*\*Git\*\* - Version control



\## Features



\- Bilingual support (English/Russian)

\- Blog posts about technical topics

\- Academic profile links (ORCID, ResearchGate, Google Scholar)

\- Skills, experience, and accomplishments sections
